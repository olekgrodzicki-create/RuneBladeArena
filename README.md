# ⚔️ Runeblade Arena

A twin-stick fantasy arena brawler with Guild Wars-style skill builds. Build your hero, equip 8 runes, survive the rooms.

## Play
Open `index.html` in a browser or host on GitHub Pages.

## Controls
| | |
|---|---|
| **Move** | WASD / Arrow keys |
| **Aim** | Mouse |
| **Basic Attack** | Left click (hold) |
| **Skills 1–8** | Keys 1–8 |
| **Pause / Rebuild** | Esc |
| **Mobile** | Left thumb = move, Right thumb = aim + shoot |

## Installing as PWA
When hosted (GitHub Pages, etc.) you'll see an **Install** prompt in your browser. Works offline after first load.

## GitHub Pages Setup
1. Push this folder to a GitHub repo
2. Go to Settings → Pages → Source: `main` branch, `/ (root)`
3. Your game is live at `https://yourusername.github.io/repo-name/`

## Features
- 32 skills across 5 lines (Blade, Pyromancy, Frost, Storm, Survival)
- Freely redistribute attribute points at any time in Pause
- Haptic feedback on mobile
- Web Audio sound effects
- SVG skill icons
- 2×4 skill bar layout
- Respawn on death (no full restart)
- Boss rooms every 5th room with checkpoints
